## Auto-Detection Patterns

When parsing SUMMARY.md and code files, look for:

### Package.json Dependencies
Extract docs links for major dependencies:
- react → https://react.dev
- next → https://nextjs.org/docs
- express → https://expressjs.com
- prisma → https://prisma.io/docs

### Import Statements
Detect external API usage:
- stripe → https://stripe.com/docs
- aws-sdk → https://docs.aws.amazon.com
- firebase → https://firebase.google.com/docs

### Environment Variables
Track which services are integrated:
- STRIPE_KEY → Stripe API
- AWS_ACCESS_KEY → AWS Services
- DATABASE_URL → Database connection

### Comments with URLs
Extract URLs from code comments:
```
// See: https://example.com/docs
// Reference: https://stackoverflow.com/...
```
